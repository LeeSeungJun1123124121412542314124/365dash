# NPS 페이지 전면 개편

## Context

NPS 페이지의 스코어카드, 차트, 필터, 데이터 테이블을 사용자 요구에 맞게 개편.

## 수정 대상 파일
- `backend/app/api/nps.py` — 스코어카드 응답에 전체값(baseline) 추가
- `frontend/src/pages/NpsPage.tsx` — 카드 6개, 차트 스크롤, 데이터 테이블 추가
- `frontend/src/lib/chartUtils.ts` — NPS 전용 초기 필터 함수 추가

---

## 1. 스코어카드 변경 (6개 카드: 전체값 3 + 필터값 3)

- **1행 (전체값)**: 매우만족 / 만족 / 보통이하 — 전체 지점 합산, 기간 전체 누적
- **2행 (필터값)**: 매우만족 / 만족 / 보통이하 — 선택 그룹/지점, 기간 전체 누적
- 카드 표시: `87.6%` (큰 글씨) + `(1,244건)` (보조 텍스트)
- 필터값 카드 제목: 선택된 대분류/중분류 이름 반영 (예: `매우만족 (병원급)`)

## 2. 기간 필터 — NPS 전용 초기값 3년

- 초기값: 3년 전 동월 ~ 현재월 (`npsDefaultFilter()`)
- FilterBar 유지, 기간 변경 가능

## 3. NPS 추이 차트

- 기본 타입: 막대 (bar)
- 24개월 초과 시 가로 스크롤 (`overflow-x-auto`)
- 범례: `기준값(전체)`, `필터값(${filterLabel})` 동적 표시

## 4. 백데이터 테이블

- 차트 아래 월별 수치 표
- 행: 전체 3행 + 필터 3행 (매우만족%, 만족%, 보통이하%)
- 필터 행 레이블: 선택된 대분류/중분류 이름으로 동적 변경
- 24개월 초과 시 가로 스크롤, 첫 열 sticky

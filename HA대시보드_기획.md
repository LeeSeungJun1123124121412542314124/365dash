# HA 대시보드 기획

---

## 1. 메인 대시보드

> 최근 6개월 데이터 메인대시보드에 표시 (전체지점만 표시)

**헤더 영역**
- 접속자명
- 지점명

### 주요 지표

| 항목 | 설명 |
|------|------|
| 참여율 (최근 6개월 평균) | n.n% |
| NPS (최근 6개월 평균) | 매우만족 n.n% / 만족 n.n% / 보통이하 n.n% |
| 칭찬현황 | 최근 6개월 평균 칭찬 개수 – n건 |
| 불만현황 | 최근 6개월 평균 불만 개수 – n건 |

### 그래프 영역
- 모바일만족도 참여율 그래프 표시
- 전지점 모바일만족도 그래프 표시
- 최근 6개월 칭찬 개수 그래프
- 최근 6개월 불만 개수 그래프

### 메뉴 구성
- 모바일만족도 참여율
- NPS
- 칭찬/불만 현황
- VOC 업로드
- 게시판
- 기타자료현황
